# carsome

# Node SetUp

UnZip "carsome-node".
- npm i
- run "node index.js"

Node will run with port 3000

# Angular SetUp

Unzip "carsome-angular".
- npm i
- ng serve -o

It will run angular app with port 4200

-- Select Date.
-- You will get list of Time slots. If you click on that it will reserve time for you.
